Cube Coloring Problem

This is a pretty extensive project I undertook just before starting college which gave me a lot of useful mathematics experience. The problem (which I believe is novel) is whether the faces of the composing unit cubes of an n x n x n cube can be colored with n different colors so that they can be rearranged into n configurations where the entire surface is colored entirely with each distinct color. This is the resulting paper I wrote.
